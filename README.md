ThinkingInJava

A beginner-friendly Java learning project with simple explanations, runnable examples, and hands-on practice — all inside the console.

Purpose

This project helps to understand core Java concepts by:

Showing a short example

Running the example

Letting the user practice inside the console

Perfect for beginners or anyone starting Java.

Project Structure
ThinkingInJava/
 ├── Main.java                 → Main menu
 ├── BasicPrograms.java        → Print, variables, arithmetic
 ├── RepeatLoops.java          → For loops
 ├── WhileLoop.java            → While loops
 ├── Logic.java                → Basic logic & decisions
 ├── Pattern.java              → Pattern printing
 ├── ChoosingConditions.java   → If-else decision making
 ├── NestedLoop.java           → Nested loops
 ├── If.java                   → If statements
 ├── IfElse.java               → If-else statements
 ├── NestedCondition.java      → Nested conditions
 ├── WebDevelopment.java       → Simple web dev example in Java
 ├── Automation.java           → Basic automation example
 ├── AppDevelopment.java       → Small app creation example
 ├── PracticeArea.java         → User practice zone
 └── README.md

How to Run
Compile all files
javac *.java

Run the main program
java Main
